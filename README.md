# Basic File Storage System

File storage system to store files into in-memory H2 database as blob, delete files and retrieve list of files through Spring Boot REST API endpoints or React client.

## Used Technologies

### Spring Boot
**Dependencies**
1. Spring Boot Web
2. Spring Data JPA
3. Lombok
4. H2 Database
5. OpenAPI UI

### React
**Packages**
1. Axios
2. Bootstrap
