# Basic File Storage System
File storage system to store and retrieve files through Spring Boot REST API and interact through React client

## Used Technologies

### Spring Boot
**Dependencies**
1. Spring Boot Web
2. Spring Data JPA
3. Lombok
4. H2 Database
5. OpenAPI UI

### React
**Packages**
1. Axios
2. Bootstrap

### Swagger

**Swagger URL:** http://localhost:8080/swagger-ui/index.html
